# Prompt  
Criar ebook de pelo menos 50 páginas, com o título "Java, do zero ao profissional". A linguagem deve ser de fácil entendimento, com foco em adolescentes no início da carreira.
Abranger os seguintes tópicos:
1. Fundamentos da Programação: Lógica, algoritmos, variáveis, tipos de dados, operadores, estruturas de controle (if/else, loops).
2. Introdução ao Java: JVM, sintaxe básica, compilação e execução de programas, "Hello World".
3. Orientação a Objetos (POO): Classes, objetos, atributos, métodos, construtores, encapsulamento, herança, polimorfismo, interfaces, abstração, pacotes.
4. Estruturas de Dados: Arrays, List, Set, Map (Collections Framework), Enum.
5. Tratamento de Erros: Exceções (try-catch-finally), Checked vs Unchecked Exceptions.
6. Entrada/Saída (I/O) e Arquivos: Manipulação de arquivos e Streams.
7. Generics e Lambdas: Tipos genéricos e programação funcional. 
8. Frameworks como Spring Boot, MVC, Actuator
9. APIs RESTful e melhores práticas
10. Flyway e outras soluções
11. Microsserviços
12. Clean Code
13. S.O.L.I.D.
14. Design Patterns
Fique à vontade para adicionar novos tópicos que achar importante.

# Adapta.org (Melhorador de Prompt)  
**"Java, do Zero ao Profissional: Um Guia Abrangente para Iniciantes"**

Crie um eBook de no mínimo 100 páginas, direcionado a adolescentes que estão iniciando sua carreira em programação, com o objetivo de proporcionar uma introdução completa à linguagem Java e conceitos relacionados. O conteúdo deve ser apresentado de maneira clara, acessível e com exemplos práticos. A IDE utilizada será o Eclipse.

**Tópicos a serem abordados:**

1. **Fundamentos da Programação**:
   - Introdução à lógica de programação e algoritmos
   - Variáveis, tipos de dados e operadores
   - Estruturas de controle (if/else, loops) com exemplos práticos

2. **Introdução ao Java**:
   - O que é JVM e sua importância
   - Sintaxe básica da linguagem Java
   - Compilação e execução de programas Java
   - Exemplo de "Hello World" e explicação detalhada

3. **Orientação a Objetos (POO)**:
   - Classes, objetos, atributos e métodos
   - Construtores e encapsulamento
   - Herança, polimorfismo e interfaces
   - Abstração e pacotes, com exemplos de implementação

4. **Estruturas de Dados**:
   - Arrays e introdução ao Collections Framework
   - List, Set e Map: características e uso adequado
   - Enum: definição e aplicações

5. **Tratamento de Erros**:
   - Exceções em Java: try-catch-finally
   - Checked vs Unchecked Exceptions: diferenças e quando usar cada uma

6. **Entrada/Saída (I/O) e Arquivos**:
   - Manipulação de arquivos em Java
   - Streams: conceito e utilização

7. **Generics e Lambdas**:
   - Tipos genéricos: definição e vantagens
   - Programação funcional com Lambdas: conceitos básicos e exemplos

8. **Frameworks e Tecnologias**:
   - Introdução ao Spring Boot: configuração e primeiros passos
   - MVC: conceito e implementação
   - Actuator: monitoramento e gerenciamento

9. **APIs RESTful e Melhores Práticas**:
   - Construção de APIs RESTful com Java
   - Princípios e práticas recomendadas para design de APIs

10. **Flyway e Outras Soluções**:
    - Controle de versão de banco de dados com Flyway
    - Outras soluções relevantes para gerenciamento de banco de dados

11. **Microsserviços**:
    - Introdução ao conceito de microsserviços
    - Vantagens e desafios na implementação

12. **Clean Code**:
    - Princípios para escrever código limpo e legível
    - Boas práticas para manutenção e evolução do código

13. **S.O.L.I.D.**:
    - Princípios de design orientado a objetos para código robusto e flexível

14. **Design Patterns**:
    - Introdução aos padrões de design mais comuns em Java
    - Exemplos de aplicação e benefícios

**Tópicos Adicionais Sugeridos:**
- Boas práticas de segurança em aplicações Java
- Testes unitários e integração com JUnit
- Utilização de ferramentas de build como Maven ou Gradle
- Introdução ao desenvolvimento de aplicações web com Java

**Requisitos de Formatação:**
- Utilizar linguagem clara e exemplos práticos para cada tópico
- Incluir ilustrações, diagramas ou código de exemplo sempre que possível
- Organizar o conteúdo de maneira lógica e progressiva, permitindo que o leitor avance gradualmente nos conceitos

**Diretrizes para a Entrega:**
- O eBook deve ser entregue em formato PDF
- Incluir sumário e numeração de páginas
- Utilizar fontes claras e tamanho de letra adequado para leitura confortável

**Critérios de Avaliação:**
- Clareza e acessibilidade do conteúdo para o público-alvo
- Cobertura abrangente dos tópicos especificados
- Qualidade dos exemplos e ilustrações utilizados
- Organização lógica e fácil navegação pelo conteúdo
